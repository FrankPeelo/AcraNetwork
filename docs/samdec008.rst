SamDec008 Documentation
**********************************

.. py:currentmodule:: AcraNetwork.SamDec008

This is a class to handle the SAM/DEC/008 USB PCM decommutator. Once programmed this will
capture ethernet packets from the SAM/DEC/008, align the to the sync word and return them as 
PCM frames
